iScatter README

Michel Wermelinger, The Open University, UK

iScatter is a library to produce interactive web-based scatterplots.
More information can be found at http://tiny.cc/iScatter

This file is part of iScatter's resource bundle, available from the iScatter site.
This is version 1.1 of the bundle.

This bundle contains the following folders:

- src: the iScatter code
- lib: the 3rd-party Guiders-JS code and style sheet needed for the guided help tour
- data: some example data sets
- doc: an example HTML file showing how to create and configure a chart within a web page
